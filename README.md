# DataVisualization

Program that will visualize data from a given csv file in a multi-dimensional plane.

Instructions:
1. Git clone the repository.
2. Execute the code by opening your local terminal.
3. Change directory (cd) to the respective folder containing the repository. 
4. Run the code by typing "python demo.py".

Notes:
Program is currently only able to visualize data in a four-dimensional plane.
